Paste your tracking IDs below. Scripts are injected automatically on every page after save & deploy. Leave a field blank to disable that tracker.

🔗 Tracking IDs
15–16 digit numeric ID from Facebook Events Manager. Loads fbevents.js and fires PageView.
Format: GTM- followed by 6–8 alphanumeric chars. Loads GTM in <head> + noscript fallback in body.
Format: G- followed by 10 alphanumeric chars. Skip if you load GA4 through GTM.
🔧 Custom Code (Advanced)
Raw HTML/JS injected into <head> on every page. Use only for trusted code — runs on all visitors.
Raw HTML/JS injected at the end of <body>. Same trust caveat applies.
